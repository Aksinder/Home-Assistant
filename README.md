# Home-Assistant

Im running HA in a vm on my server.

## Hardware
* Server: 
  * Supermicro X9SCA-F + Intel Xeon E3-1235 + 32 GB DDR3 ECC RAM with a bunch of disks.
* Raspberry PI3 (Located in the middle of the house)
  * Aeotec Z-stick Gen5 
  * RFXcom
* Harmony Hub
* Chromecast
* Chromecast audio x3
* Google Home Mini
* A bunch of 433mhz plug-in outlets.
